# Live View of Covid 19 Data

## Installation

1. Install Elixir dependencies:

    ```sh
    mix deps.get
    ```

2. Create the database:

    ```sh
    mix ecto.setup
    ```

3. Fire up the Phoenix endpoint:

    ```sh
    mix phx.server
    ```

4. Visit [`localhost:4000`](http://localhost:4000/)
